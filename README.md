# Скрипты Bash

## Описание
Набор bash-скриптов для выполнения математических операций, включая LU-декомпозицию матриц.

## Принципы работы

### 1) LU декомпозиция

Скрипт выполняет LU-разложение матрицы на нижнетреугольную (L) и верхнетреугольную (U) матрицы. Приходят 4 аргумента: путь к входной матрице(csv), путь к выходным матрицам L и U в формате csv и точность разложения(знаков после запятой)


**Формат вызова:**
./LU_decomposition.sh input.csv lower.csv upper.csv 7

### 2) Фильтрация логов

Скрипт собирает все данные из файлов с расширением .log. Корректный формат логов задан регулярным выражением '[<дескриптор>]_<дд:мм:гггг>_<id>_<сообщение>'. Выводятся первые lines логов в случае up и последние lines в случае down (Если корректных логов меньше выводятся все). Логи отсортированы по id
